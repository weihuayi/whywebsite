Pdf 图片的剪裁

当手头的 Pdf 图片边上有很多空白时, 可以用 Texlive 中的 pdfcrop 命令剪裁

首先要安装 Texlive 的辅助程序包, 里面有很多有用的小程序

$ sudo apt-get install texlive-extra-utils

$ pdfcrop  fig.pdf
