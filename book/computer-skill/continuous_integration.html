Travis CI

连续集成测试 (Continuous integration, CI): 集成测试环境

  1. 你可以用 GitHub 的帐户登录
  2. 选择你要做集成测试的仓库
  3. 添加 .travis.yml 文件
  4. 对本地仓库做修改, 并投送到 GitHub 仓库
  5. Travis CI 默认当仓库有新的 Push的时候构建你的软件
